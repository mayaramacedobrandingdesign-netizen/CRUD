**Sistema de Controle de Itens (CRUD)**

Este projeto consiste em um sistema simples de controle de itens, desenvolvido com React e JavaScript, que permite realizar operações básicas de gerenciamento de dados utilizando o conceito de CRUD (Create, Read, Update e Delete).

A aplicação permite que o usuário adicione, visualize, atualize e remova itens de uma lista de forma dinâmica através de uma interface gráfica.

**Funcionalidades**

O sistema implementa as seguintes operações:

**Adicionar item**

Permite inserir um novo item na lista através de um campo de texto.

**Visualizar itens**

Os itens adicionados são exibidos automaticamente em uma lista organizada.

**Atualizar item**

O usuário pode selecionar um item da lista para editar seu conteúdo.

**Remover item**

Permite excluir um item específico da lista.

**Tecnologias utilizadas**

-JavaScript
-React
-CSS
-HTML

O gerenciamento de dados é feito utilizando arrays e o estado do React (useState).

**Objetivo do projeto**

Este projeto foi desenvolvido como atividade prática para aplicar conceitos fundamentais de programação, incluindo:

-Funções
-Arrays
-Estruturas condicionais
-Manipulação de dados
-Conceito de CRUD
-Desenvolvimento de interfaces com React
